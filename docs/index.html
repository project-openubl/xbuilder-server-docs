Introduction

XBuilder Server is a Microservice that can create XML files based on the Universal Bussiness Language (UBL). It is based on XBuilder

Features#

XBuilder Server is based on XBuilder so it offers the same core funtionalities. However, XBuilder Server offers additional features that allows you to deploy it across different cloud platforms or bare-metals.
